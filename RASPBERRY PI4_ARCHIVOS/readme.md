# Descripción
En esta carpeta se encuentran los archivos de simulación separados como .py, el archivo config corresponde al código necesario para configurar el token de UBIDOTS, variables de
UBIDOTS y serial, en el archivo funciones se tienen todas los prototipos necesarios para ejecutar la conexión del Raspberry con UBIDOTS; en el archivo main se ejecutan las
funciones creadas dentro del bucle.
# Indicaciones
El archivo .pdsprj tiene la incluído todos los códigos para ejecutarse, sin embargo se incluye la división en archivos .py de modo que se puede hacer un nuevo esquemático
e incluir estos archivos para ejecutarlo sin problemas, por lo que se dejan ambas opciones para el lector.
